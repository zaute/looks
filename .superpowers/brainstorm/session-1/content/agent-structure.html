How should the agents work together?

Pick the structure that feels most useful to you day-to-day

A

Standalone Specialists

Each agent works independently. You go to the haircut agent for hair advice, skincare agent for skin, etc. Simple and focused.

B

One Master Agent + Specialists

A single "Personal Image Director" agent takes your photo or description, then coordinates the specialists and delivers a unified action plan — one conversation, complete output.

C

Weekly Check-in System

A routine-focused agent that runs a weekly review — tracks progress, adjusts your skincare/fitness/wardrobe plans over time, and gives you a prioritised "this week" action list.
